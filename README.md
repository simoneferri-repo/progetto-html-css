# Progetto per l'esame di "PROGETTAZIONE E PRODUZIONE MULTIMEDIALE"
L'elaborato prevede la riproduzione della home page del sito web assegnato, dove con riproduzione si intende solo il layout costruito con l'utilizzo di CSS, HTML5.

Per la realizzazione del progetto sono state utilizzate le seguenti tecnologie:
- HTML 5
- CSS
- Bootstrap 5
- Javascript (solo per lo switch della classe css nel menu principale al cambio di risoluzione)

**Nota**: per simulare l'effetto "up/down" della testata allo scroll bidirezionale della pagina (nel sito originale realizzato con javascript) sono state utilizzate le "CSS container scroll-state queries". Queste sono supportate solo da Chrome ed Edge, su Firefox e Safari la testata viene visualizzata con un effetto fixed.
